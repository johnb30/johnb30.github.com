PresidentParser

Lafleur, McKendon and John Beieler. 2012. “PresidentParser: Automated Mark-up Utility.” The Political Methodologist. 19:2.

  • Click here for the copy of The Political Methodologist featuring the article describing PresidentParser.

PresidentParser is a software utility designed to aid in the preparation of spee ch acts for coding in the Profiler Plus software program provided by Social Science Automation. Despite the name, which grew out of the original use of the program, PresidentParser is able to parse any type of file that has a consistent pattern. Typically, when preparing a speech act, a researcher would have to manually mark-up a speech act in order to ignore questions from reporters, etc., that did not constitute an individual of interest’s spontaneous response to a question. PresidentParser automates this process by ignoring questions, comments in brackets, question headers, and the introduction to a conference given by the President that is not the result of a question being asked. The program makes use of text dictionaries to identify patterns within a document that allow for the correct insertion of the markup. The program was written by McKendon Lafleur of Louisiana State University, with myself providing a small amount of assistance. McKendon can be contacted at McKendon [at] acm [dot] org. The download for the program is below. Included is a README file, which explains in detail how to use the program. In addition, the program can be downloaded from github, which provides access to the source in the event that anyone would like to modify or contribute to the program. If you do modify the program for your own use, please notify us since we would be interested to see what situations the program is being used in, and please cite as indicated in the README text.

py_apsrtable

py_apsrtable aims to be the python implemenation of the R package apsrtable.

py_asprtable takes as its input a model, or set of models, generated by the Python packages statsmodels and pandas, and outputs a LaTeX table formatted in a manner consistent with common political science formatting. The formatting is also nice for other applications.

Options for the models includes standard errors, two-tailed p values, and one-tailed p values.

Code and more information can be found on github. An example of the final PDF output can be seen here.
